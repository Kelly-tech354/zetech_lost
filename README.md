
📌 Campus Lost and Found Management System
📖 Project Description

The Campus Lost and Found Management System is a web-based application designed to simplify the reporting, tracking, and recovery of lost items within a university environment. The system allows users to report lost or found items, while administrators manage records and verify ownership before releasing items. It aims to improve efficiency, transparency, and accountability in handling lost property.

🎯 Objectives
To provide a centralized platform for reporting lost and found items
To improve tracking and retrieval of misplaced belongings
To enhance communication between users and administrators
To ensure secure and organized record management
👥 Users
Students – Report lost items and claim recovered ones
Staff – Report and manage found items
Administrator – Verify ownership, manage records, and oversee system operations
⚙️ Features
User registration and login system
Report lost and found items
Search and filter items
Item status tracking
Notification system for matched items
Admin verification before item release
Secure database for record storage
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP
🚀 Installation Guide
Install XAMPP
Start Apache and MySQL from XAMPP control panel
Copy the project folder into the htdocs directory
Import the database file into phpMyAdmin

Open your browser and go to:

https://localhost/zetech
